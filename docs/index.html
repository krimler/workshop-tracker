Workshop GPT

Every ML/CS workshop with its deadline, in one place.

There is no single list of academic workshops. Each one runs its own call for papers, with its own deadline, on its own page, so finding them means checking conferences one by one. Workshop GPT collects them in one place: the workshop, its parent conference, the submission deadline, and the location. The data is rebuilt daily from public sources.

What's inside

Workshops across nine research areas (AI/ML, theory, security, systems, networking, databases, software engineering, distributed systems, biomedical), each mapped to its parent conference (NeurIPS, ICML, ICLR, COLM, CVPR, ACL, and more) with deadline and location. An AI assistant answers plain-language questions like "LLM-agent workshops closing soonest" using only the tracked data.
